Montagem Experimental 02
======
O objetivo desta montagem é mostrar uma contagem de 0-9 em um display 7-segmentos, que incremente o valor do número mostrado a cada 1 segundo. 

<center>
<br/>
<img src='/images/2ele031/lab1/display.png'>
<p>Figura 5-Esquemático de Display 7-segmentos Câtodo comum e Anodo Comum</p>
</center>

Implemente o circuito da Figura 06 com um display 7-segmentos Catodo Comum. Considere a pinagem do display conforme o mostrado na Figura 05.

<center>
<br/>
<img src='/images/2ele031/lab1/ligacao_display.png'>
<p>Figura 6-Acionamento de um display Catodo comum utilizando o Arduino</p>
</center>

O código base conta com três funções para a manipulação do display 7-segmentos:

* `setDisplayClear()`: Desliga todos os segmentos do display.
* `setDisplayCount()`: Executa uma contagem em ordem crescente de 0-9 no display.
* `setDisplayDigit(int)`: Realiza o acionamento dos segmentos do display de acordo com o parâmetro de entrada.

Você deve utilizar estas funções para realizar este experimento e os outros a seguir. Contudo, é necessário realizar a implementação destas funções, uma vez que estão incompletas.

> Pode-se alterar os pinos utilizados para o acionamento dos display 7-segmentos.

Questões
=====

* Explique o funcionamento do display 7 segmentos.
* Descreva seu código e as soluções implementadas.
