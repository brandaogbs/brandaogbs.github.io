Montagem Experimental 03
=====

O objetivo da montagem experimental 3 é adicionar um segundo display 7-segmentos, tornando possível mostrar dois digitos. 

Todavia, se analisarmos a placa de desenvolvimento veremos que não temos pinos digitais suficientes para repetirmos duas vezes a lógica da Montagem 2. Assim, será necessário realizar a multiplexação dos displays.

De forma simplificada o processo de multiplexação significa utilizar os mesmos pinos de controle dos 7 LEDs do display comandando assim ambos os displays. Com a adição de uma lógica (feita por transistores) podemos acionar cada um dos displays separadamente.

Se o acionamento for rápido o suficiente nosso olho não é capaz de detectar que somente um dos displays está ligado, e veremos o número em ambos os displays.

Adapte o circuito da Montagem 2 agora para o circuito mostrado na Figura 8. A seleção de qual display deve estar ligado é feito por dois transistores NPN atuando como chave.

Implemente um código que mostre dois números distintos (por exemplo, 69), o número 6 no primeiro display e o número 9 no segundo display. Para tanto, você deve ficar alternando em uma frequência alta qual display está acionado. 

A lógica anterior pode ser feita com o seguinte procedimento

1.  Desligue o Pino 13;
2.  Acione os Pinos 2-8 conforme necessário para mostrar o digito 6;
3.  Acione o Pino 12;
4.  Aguarde 1ms
5.  Desligue o Pino 12;
6.  Acione os Pinos 2-8 conforme necessário para mostrar o digito 9;
7.  Acione o Pino 13;
8.  Aguarde 1ms
9.  Volte a etapa 1
