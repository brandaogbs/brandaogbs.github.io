A Disciplina
=====
* O objetivo da disciplina de Projeto Integrador é realizar a fixação e reforçar os conceitos aprendidos durante o semestre em um projeto prático.
* Serão apresentados os conceitos básicos sobre as ferramentas utilizadas para o desenvolvimento. Porém, não é o intuito da disciplina o aprofundamento ou as apresentações teóricas envolvidas, ficando este a cargo das equipes.
* A avaliação da disciplina é dada através do desenvolvimento do projeto, desta forma em cada aula deverá ser entregue um relatório que resume as atividades desenvolvidas desde o último encontro. Além disso, deve ser realizada uma breve apresentação das funcionalidades implementadas, compondo assim a nota total da disciplina.

A Proposta
=====
* Os alunos deverão realizar a implementação de um jogo em plataforma 2D.
* Este jogo deve envolver diretamente conceitos de física, cálculo e algebra linear.
* A linguagem de programação utilizada no desenvolvimento será obrigatóriamente `Python3`.
* Além disso, será utilizada uma *engine* gráfica para o desenvolvimento do jogo.

O Tema
=====
* O tema/jogo deve ser escolhido pela equipe.
* O tema será oficializado no segundo encontro, onde os alunos devem entregar um documento descritivo da ideia.
* No documento descritivo deve constar também uma breve separação das atividades que devem ser realizadas em cada uma das semanas.

As Equipes
=====
* As equipes serão formadas por três alunos.
* Os alunos devem ser da mesma turma.

As Aulas
=====
* As aulas serão quinzenais.
* Nas primeiras aulas serão reapresentados conceitos básicos de linguaguem de programação.
* As aulas seguintes serão utilizadas para o desenvolvimento do projeto, bem como para a apresentação das equipes.
* A aula terá seu tempo dividido para que cada equipe faça a apresentação do relatório de atividades.

A Avaliação
=====
* O critério de avaliação será baseado nas entregas quinzenais.
* Cada entrega será de mesmo peso e a soma de todas as entregas será a média final.
* A não apresentação dos desenvolvimentos quinzenais implica na não obtenção da respectiva nota.

As Entregas
=====
* As entregas quinzenais serão compostas por dois items:
    * Um breve relatório descritivo das atividades.
    * Apresentação dos resultados obtidos e dos problemas enfrentados.
* Os relatórios devem ser enviados para o email `brandaogbs@gmail.com` até o dia anterior a aula.
* O modelo do relatório será fornecido pelo professor.
