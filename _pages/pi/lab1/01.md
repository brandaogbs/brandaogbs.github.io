Montagem Experimental 1 – Acionamento de uma Lâmpada
======

O objetivo desta montagem é acionar uma Lâmpada conectada a Rede Elétrica a partir de um Chave conectada a um microcontrolador (neste caso o Arduino). Este circuito é empregado em sistemas de automação residencial, para ligar e desligar eletrodomesticos.

O circuito que deve ser montado, estada disposto na Figura 7. Este é responsável por acionar uma lâmpada comum 127 Vac a partir do sinal de uma chave que está conectada ao microcontrolador. 

Monte este circuito (com Rb = 1k ohm) e elabore o código no microcontrolador para detectar o nível lógico do botão e então comandar a saída responsável por ligar/desligar a lâmpada. 
> O código deve ser inserido dentro da função `isButtonPressed()`.
