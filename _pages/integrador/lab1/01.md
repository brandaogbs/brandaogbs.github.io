Montagem Experimental 1–Acionamento de uma Lâmpada
======

O objetivo desta montagem é acionar uma Lâmpada conectada a Rede Elétrica a partir de um Chave conectada a um microcontrolador (neste caso o Arduino).

O circuito que deve ser montado, mostrado na figura 7, é responsável por acionar uma lâmpada comum 127 Vac a partir do sinal de uma chaveque está conectada ao arduino. Este circuito pode ser facilmente empregado em sistemas de automação residencial.

Monte este circuito (com Rb = 1k ohm) e elabore o código em Arduino para detectar o nível lógico do botão e então comandar a saída responsável por ligar/desligar a lâmpada.

Explique no relatório o funcionamento detalhado deste circuito, incluindo o cálculo do resistor de Rb.
